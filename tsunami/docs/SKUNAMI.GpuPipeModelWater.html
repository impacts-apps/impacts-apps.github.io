SKUNAMI. GpuPipeModelWater

new GpuPipeModelWater(options)

GPU height field water based on the hydrostatic pipe model ("Fast Hydraulic Erosion Simulation and Visualization on GPU", Xing Mei, Philippe Decaudin and Bao-Gang Hu, Pacific Graphics 2007)
Parameters:
Name Type Description
options object Options
Properties
Name Type Argument Default Description
renderer THREE.WebGLRenderer Three.js WebGL renderer
scene THREE.Scene Three.js scene
mesh THREE.Mesh Three.js mesh for sculpting
size number Size of mesh
res number Resolution of mesh
dampingFactor number Damping factor for the sim
multiSteps number <optional>
 1 Number of full steps to take per frame, to speed up some of algorithms that are slow to propagate at high mesh resolutions
terrainTexture THREE.WebGLRenderTarget <optional>
 null Terrain texture that creates a base height which affects the water
initialWaterHeight number <optional>
 0 Initial height of water that is above the terrain
density number <optional>
 1000 Density of water (kg per cubic metres)

source(position, amount, radius)

Sources water into the simulation and causes water level to rise. This is different from disturb which is meant to only create small ripples on the water surface. A negative amount will create a sink which removes water from the simulation and causes water level to fall.
Parameters:
Name Type Description
position THREE.Vector3 World-space position to source at
amount number Amount of water to source. A negative amount removes water from the system.
radius number Radius of water to source
